## 📚 Roadmap & Syllabus Breakdown

### 🔹 1. Foundational SQL (Beginner)
* **Data Definition Language (DDL):** `CREATE`, `ALTER`, `DROP`, `TRUNCATE`
* **Data Manipulation Language (DML):** `SELECT`, `INSERT`, `UPDATE`, `DELETE`
* **Basic Filtering & Sorting:** `WHERE`, `AND/OR/NOT`, `IN`, `BETWEEN`, `LIKE` (Wildcards), `ORDER BY`, `LIMIT`
* **Aggregation Basics:** `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`, `GROUP BY`, `HAVING`

### 🔹 2. Intermediate Querying & Database Design
* **Relational Joins:** `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `FULL OUTER JOIN`, `CROSS JOIN`, and Self-Joins
* **Subqueries & Set Operators:** Scalar, Column, and Correlated Subqueries; `UNION`, `UNION ALL`, `INTERSECT`, `EXCEPT`
* **Built-in Functions:** String Manipulation (`CONCAT`, `SUBSTRING`, `REPLACE`), Date/Time Arithmetic, and Numeric Functions
* **Conditional Logic:** Masterful use of `CASE WHEN ... THEN ... ELSE END`

### 🔹 3. Advanced Data Analytics (Expert)
* **Window Functions:** Analytics operations using `OVER()`, `PARTITION BY`, and `ORDER BY`
  * *Ranking:* `ROW_NUMBER()`, `RANK()`, `DENSE_RANK()`
  * *Value:* `LAG()`, `LEAD()`, `FIRST_VALUE()`, `LAST_VALUE()`
  * *Aggregates:* Running totals and moving averages
* **Common Table Expressions (CTEs):** Writing clean, modular, and readable queries using non-recursive and recursive `WITH` clauses
* **Query Optimization & Performance Tuning:** Analyzing Execution Plans (`EXPLAIN`), understanding indexing strategies (Clustered vs. Non-Clustered), and minimizing nested loop costs
